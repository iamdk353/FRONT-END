# **Task 4: Responsive Design & Media Queries (Days 10-12)**

## Deadline in 2 days

**Goal:** Create mobile-first, responsive websites

## Task 1: Learn mobile-first design principles

#### Focus on: Designing for the smallest screens first and then progressively enhancing for larger screens.

[Mobile First Design Reference](https://www.w3schools.com/css/css_rwd_intro.asp)
[Responsive Web Design Basics](https://web.dev/responsive-web-design-basics/)
[Mobile-First CSS Guide](https://css-tricks.com/logic-in-media-queries/)
[Viewport Meta Tag Guide](https://developer.mozilla.org/en-US/docs/Web/HTML/Viewport_meta_tag)

## Task 2: Master media queries and breakpoints

#### Focus on: Syntax of media queries (`@media`), common breakpoints for devices (phones, tablets, desktops), and how to apply different CSS rules based on screen characteristics.

[CSS Media Queries Reference](https://www.w3schools.com/css/css_rwd_mediaqueries.asp)
[Media Queries Guide](https://css-tricks.com/a-complete-guide-to-css-media-queries/)
[Common Breakpoints Reference](https://getbootstrap.com/docs/5.0/layout/breakpoints/)
[MDN Media Queries](https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries/Using_media_queries)

## Task 3: Understand responsive units

#### Focus on: `rem`, `em` for scalable text; `vw` (viewport width), `vh` (viewport height) for sizing relative to the viewport; `%` for relative sizing.

[CSS Units Reference](https://www.w3schools.com/css/css_units.asp)
[Viewport Units Guide](https://css-tricks.com/fun-viewport-units/)
[Em vs Rem Guide](https://webdesign.tutsplus.com/tutorials/comprehensive-guide-when-to-use-em-vs-rem--cms-23984)
[Responsive Typography](https://css-tricks.com/responsive-typography-using-viewport-units/)
