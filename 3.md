# **Task 3: CSS Flexbox & Grid (Days 7-9)**

## Deadline in 2 days

**Goal:** Master modern CSS layout systems

## Task 1: Learn Flexbox properties and use cases

#### Focus on: Flex container properties (display, flex-direction, justify-content, align-items, flex-wrap) and flex item properties (flex-grow, flex-shrink, flex-basis, order, align-self). Ideal for 1D layouts.

[CSS Flexbox Reference](https://www.w3schools.com/css/css3_flexbox.asp)
[Complete Guide to Flexbox](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)
[MDN Flexbox Guide](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox)
[Flexbox Froggy Game](https://flexboxfroggy.com/)

## Task 2: Understand CSS Grid for 2D layouts

#### Focus on: Grid container properties (display, grid-template-columns, grid-template-rows, grid-gap) and grid item properties (grid-column, grid-row). Ideal for complex 2D page layouts.

[CSS Grid Reference](https://www.w3schools.com/css/css_grid.asp)
[Complete Guide to CSS Grid](https://css-tricks.com/snippets/css/complete-guide-grid/)
[MDN CSS Grid Guide](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Grids)
[Grid Garden Game](https://cssgridgarden.com/)

## Task 3: Practice layout combinations and choosing the right tool

#### Focus on: When to use Flexbox vs Grid, combining both systems, responsive design with modern layouts.

[Flexbox vs Grid Comparison](https://css-tricks.com/quick-whats-the-difference-between-flexbox-and-grid/)
[Using Grid and Flexbox Together](https://css-tricks.com/css-grid-replace-flexbox/)
