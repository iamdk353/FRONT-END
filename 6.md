# **Task 6: DOM Manipulation & Events (Days 16-18)**

## Deadline in 4 days

**Goal:** Make websites interactive with JavaScript

## Task 1: Select and manipulate DOM elements

#### Focus on: `getElementById`, `getElementsByClassName`, `getElementsByTagName`, `querySelector`, `querySelectorAll`; changing content (`innerHTML`, `textContent`), attributes (`setAttribute`, `getAttribute`), and styles.

[DOM Selector Methods Reference](https://www.w3schools.com/js/js_htmldom_elements.asp)
[DOM Content Manipulation](https://www.w3schools.com/js/js_htmldom_html.asp)
[DOM Attributes Reference](https://www.w3schools.com/js/js_htmldom_attributes.asp)
[DOM Styling Reference](https://www.w3schools.com/js/js_htmldom_css.asp)
[MDN DOM Selection Guide](https://developer.mozilla.org/en-US/docs/Web/API/Document_object_model/Locating_DOM_elements_using_selectors)

## Task 2: Handle user events

#### Focus on: `addEventListener`, `removeEventListener`; common event types (`click`, `mouseover`, `mouseout`, `keydown`, `keyup`, `submit`, `focus`, `blur`, `change`, `load`). Understanding event object, `this` keyword in event handlers.

[JavaScript Events Reference](https://www.w3schools.com/js/js_events.asp)
[addEventListener Reference](https://www.w3schools.com/js/js_htmldom_eventlistener.asp)
[Event Object Reference](https://www.w3schools.com/js/js_htmldom_events.asp)
[MDN Event Handling Guide](https://developer.mozilla.org/en-US/docs/Web/Events/Event_handlers)
[Event Types Reference](https://developer.mozilla.org/en-US/docs/Web/Events)

## Task 3: Create dynamic content and interactions

#### Focus on: Creating new elements (`createElement`), appending/removing elements (`appendChild`, `removeChild`, `insertBefore`), traversing the DOM (parentNode, children, nextElementSibling).

[DOM Creation Methods](https://www.w3schools.com/js/js_htmldom_nodes.asp)
[DOM Navigation Reference](https://www.w3schools.com/js/js_htmldom_navigation.asp)
[MDN DOM Manipulation](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Client-side_web_APIs/Manipulating_documents)
[Element Methods Reference](https://developer.mozilla.org/en-US/docs/Web/API/Element)
