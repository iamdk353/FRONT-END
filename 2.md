# **Task 2: CSS Basics & Box Model (Days 4-6)**

## Deadline in 2 days

**Goal:** Understand CSS fundamentals and layout principles

## Task 1: Master selectors, properties, and values

#### Focus on: Type, class, ID selectors, attribute selectors, pseudo-classes, pseudo-elements; common CSS properties for text, color, background.

[CSS Selectors Reference](https://www.w3schools.com/css/css_selectors.asp)
[CSS Pseudo-classes Reference](https://www.w3schools.com/css/css_pseudo_classes.asp)
[CSS Pseudo-elements Reference](https://www.w3schools.com/css/css_pseudo_elements.asp)
[CSS Text Properties Reference](https://www.w3schools.com/css/css_text.asp)
[CSS Background Properties Reference](https://www.w3schools.com/css/css_background.asp)

## Task 2: Learn the box model

#### Focus on: Margin, border, padding, and content area; understanding `box-sizing: border-box;`.

[CSS Box Model Reference](https://www.w3schools.com/css/css_boxmodel.asp)
[Box-sizing Property Reference](https://www.w3schools.com/css/css3_box-sizing.asp)
[MDN Box Model Guide](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/The_box_model)

## Task 3: Practice positioning

#### Focus on: Static, relative, absolute, fixed, and sticky positioning; understanding the containing block and z-index.

[CSS Position Reference](https://www.w3schools.com/css/css_positioning.asp)
[Z-index Reference](https://www.w3schools.com/css/css_z-index.asp)
[MDN Positioning Guide](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Positioning)
