# **Task 5: JavaScript Fundamentals (Days 13-15)**

## Deadline in 4 days

**Goal:** Learn core JavaScript concepts and syntax

## Task 1: Variables, data types, operators, and control structures

#### Focus on: `let`, `const`, `var`; primitive types (string, number, boolean, null, undefined, symbol, bigint) and objects; arithmetic, comparison, logical operators; `if/else`, `switch`, `for`, `while` loops.

[JavaScript Variables Reference](https://www.w3schools.com/js/js_variables.asp)
[JavaScript Data Types Reference](https://www.w3schools.com/js/js_datatypes.asp)
[JavaScript Operators Reference](https://www.w3schools.com/js/js_operators.asp)
[JavaScript Conditions Reference](https://www.w3schools.com/js/js_if_else.asp)
[JavaScript Loops Reference](https://www.w3schools.com/js/js_loop_for.asp)
[MDN JavaScript Guide - Grammar and Types](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Grammar_and_types)

## Task 2: Functions, scope, and hoisting

#### Focus on: Declaring and invoking functions (function declarations, expressions, arrow functions); understanding global, local, block scope; how hoisting affects variable and function declarations.

[JavaScript Functions Reference](https://www.w3schools.com/js/js_functions.asp)
[JavaScript Scope Reference](https://www.w3schools.com/js/js_scope.asp)
[JavaScript Hoisting Reference](https://www.w3schools.com/js/js_hoisting.asp)
[MDN Functions Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions)
[Arrow Functions Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Functions/Arrow_functions)

## Task 3: Arrays and objects manipulation

#### Focus on: Creating and accessing array elements/object properties; common array methods (push, pop, shift, unshift, map, filter, reduce); object methods and iteration.

[JavaScript Arrays Reference](https://www.w3schools.com/js/js_arrays.asp)
[JavaScript Array Methods Reference](https://www.w3schools.com/js/js_array_methods.asp)
[JavaScript Objects Reference](https://www.w3schools.com/js/js_objects.asp)
[MDN Array Methods](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)
[Object Iteration Methods](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object)
